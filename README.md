Controle Financeiro API

API REST desenvolvida com Spring Boot para gerenciamento de finanças pessoais.

O projeto permite cadastrar usuários, contas, categorias e transações, atualizando automaticamente o saldo das contas conforme entradas e saídas.

Tecnologias
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- Lombok

Funcionalidades
- CRUD de usuários
- CRUD de contas
- CRUD de categorias
- CRUD de transações
- Atualização automática de saldo
- Tratamento global de exceções
- Validação com Bean Validation
